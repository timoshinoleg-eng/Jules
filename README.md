# 🤖 FreeModel Auto-Coder — AI-агент для GitHub

Автономный AI-агент через **FreeModel.dev**, который круглосуточно работает с вашим репозиторием: пишет код, создаёт Pull Request'ы, рефакторит и фиксит баги.

> Поддерживает **Claude** (через `cc.freemodel.dev`) и **GPT** (через `api.freemodel.dev`) — переключается одной переменной.

---

## ⚡ Что умеет агент

- 🔍 **Сканирует код** — ищет TODO/FIXME, уязвимости, костыли
- ✍️ **Пишет код** — реализует фичи, рефакторит, добавляет тесты
- 🔄 **Создаёт PR** — оформляет изменения в отдельную ветку с описанием
- 🛠️ **Автофикс** — если CI падает, анализирует логи и предлагает исправления
- ⏰ **Работает 24/7** — запускается по расписанию или на события

## 🚀 Быстрый старт

### 1. Добавь секреты в репозиторий

Перейди в **Settings → Secrets and variables → Actions** и добавь:

| Секрет | Значение |
|--------|----------|
| `FREEMODEL_API_KEY` | Твой API-ключ от FreeModel (начинается с `fe_oa_...`) |
| `GITHUB_TOKEN` | Не нужно создавать! GitHub даёт его автоматически в Actions |

> 💡 Получить ключ: [freemodel.dev/dashboard/keys](https://freemodel.dev/dashboard/keys)

### 2. Скопируй файлы из этого репозитория

```
.github/workflows/claude-agent.yml
scripts/agent.py
scripts/test_connection.py
```

### 3. Проверь локально (опционально но рекомендуется)

Установи переменные и запусти тест:

```bash
export FREEMODEL_API_KEY="fe_oa_..."
export GITHUB_TOKEN="ghp_..."
export REPO_FULL_NAME="username/repo"
export API_TYPE="anthropic"  # anthropic = Claude, openai = GPT
python scripts/test_connection.py
```

Тест проверит:
- ✅ Подключение к FreeModel API
- ✅ Авторизацию в GitHub
- ✅ Доступ к репозиторию и правам на PR

### 4. Запусти вручную на GitHub

**Actions → AI Agent (FreeModel) → Run workflow**

При запуске можно выбрать:
- **Режим**: `auto_todo`, `refactor`, `bugfix`, `review`
- **API тип**: `anthropic` (Claude) или `openai` (GPT)
- **Модель**: оставь пустым для авто-выбора, или укажи конкретную

## 🎛 Настройка моделей

### Claude (через cc.freemodel.dev)

Выбирай `API_TYPE = anthropic` при запуске workflow (или в env):

| Модель | Описание |
|--------|----------|
| `claude-opus-4-20250514` | Самая умная, для сложных задач |
| `claude-sonnet-4-20250514` | Баланс скорости и качества |
| `claude-haiku-4-20250514` | Быстрая и дешевая |

### GPT (через api.freemodel.dev)

Выбирай `API_TYPE = openai`:

| Модель | Описание |
|--------|----------|
| `gpt-5.5` | Флагман |
| `gpt-5.4` | Баланс цены и качества |
| `gpt-5.4-mini` | Дешёвая |
| `gpt-5.3-codex` | Специализирована под код |

### Кастомная модель

В `scripts/agent.py` или через env `MODEL`:

```python
MODEL = "claude-opus-4-20250514"   # или любая другая доступная
```

## 📊 Расходы API

FreeModel даёт **$300 бесплатных кредитов** при регистрации.

Для экономии:
- Используй `claude-haiku` или `gpt-5.4-mini` для простых задач
- Уменьши `MAX_FILES_TO_SCAN` в `agent.py`
- Увеличь интервал в cron (например, раз в сутки)
